TODO APPLICATION

This Todo application is built to help users manage daily tasks efficiently by allowing them to add, update, and delete tasks in a simple and user-friendly interface.
The project was developed to practice core programming concepts, improve logical thinking, and understand how task-based applications work in real-world scenarios. 
It focuses on clean UI design and smooth task management, making it useful for organizing everyday activities while strengthening fundamental development skills.

Features

-Add new tasks

-Edit existing tasks

-Delete tasks

-Mark tasks as completed

-Filter tasks by status (All / Completed / Pending)

-Clean and responsive UI
